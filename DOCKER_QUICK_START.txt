Problem: "docker-compose: command not found"
Solution: Install Docker Desktop (includes Docker Compose)

Problem: "MongoDB connection error"
Solution:
  1. Check Docker is running: docker-compose ps
  2. Check logs: docker-compose logs mongodb
  3. Restart: docker-compose restart mongodb

Problem: Port 27017 already in use
Solution: Stop other MongoDB or change port in docker-compose.yml

Problem: Can't start container
Solution: docker-compose down -v (reset)
          docker-compose up -d (start fresh)
